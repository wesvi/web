# Markdown

## Zettel
    - Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，可用于在纯文本文档中添加格式化元素。
    - Markdown文件的后缀名是“.md”
    - Markdown由John Gruber于2004年创建。
    - Markdown如今已成为世界上最受欢迎的标记语言之一。
	- 专注于文字内容；
	- 纯文本，易读易写，可以方便地纳入版本控制；
	- 语法简单，没有什么学习成本，能轻松在码字的同时做出美观大方的排版。
