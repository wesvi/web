# Github

- GitHub是一个基于Git的用于版本控制和协作的代码托管平台
- Github允许您和其他人随时随地协同处理项目
- Github 由Chris Wanstrath, PJ Hyett 与Tom Preston-Werner三位开发者在2008年4月创办
- 登陆网址：[Github](https://github.com/)
